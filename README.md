Space Attack is a Python-based 2D arcade game that focuses on real-time navigation and survival. A spacecraft dodging meteors, static planetary obstacles, and hostile alien entities.
Integrated with SQlite database for tracking and storing player performance.

1. Clone the Repository:
   
git clone https://github.com/arpita10678/space-attack.git

cd space-attack

2. Create a Virtual Environment
python -m venv venv

3. Activate the Virtual Environment

Windows CMD:

venv\Scripts\activate.bat


4. Install Dependencies
pip install -r requirements.txt

5. Run the Game
python main.py
